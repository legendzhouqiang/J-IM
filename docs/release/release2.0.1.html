先庆祝一下t-io被OSC评为最有价值开源项目

t-io最初开源的时候，是被喷到不行的，譬如“小猫小狗也出来写框架了”、“这代码看了几行就看不下去了”、“我随便压压，t-io网站就挂了，这能怪我？”，“垃圾，连DDOS都防不住”，一腔热情几乎被瞬间浇灭，甚至一度发布了闭源通告（现在码云上的捐赠通道全部关闭了，就是作者在闭源通告期间做的准备工作），不过事后剧情反转，一群t-io真实用户极力挽留，OSC官方不受谣言影响继续公正地支持t-io，t-io才没被夭折！所以要面对质疑时，最好的回应其实是做强自己，就如中医中的一句话“正气内存，邪不可干”！
