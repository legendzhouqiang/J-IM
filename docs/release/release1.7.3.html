1.7.1版本的新闻中解释了【不仅仅是百万级即时通讯框架】中【不仅仅】的含义，这个版本就是来兑现这个【不仅仅】的

  • 特别说明

    t-io官网备案完成: http://t-io.org
    • 主机已经从阿里云的1M,1核,1G迁到由杭州虎牛科技有限公司提供的主机（感谢该公司对开源事业的支持），
    • 目前运行在tio-httpserver下，大家可以f12一把看看tio-httpserver有什么不一样的，tio-httpserver正式版还没有发布，有问题欢迎反馈！
    • 官网目前是作者自己套的界面，已经在找专业UI重新设计了，改版可期
    • 官网的核心部分已经开源，有兴趣可以去看看：https://git.oschina.net/tywo45/tio-site

  • 更新说明

    • bugfix: java aio WriteCompletionHandler.completed()在各平台表现不一致----centos下并不等发完才回调，而是发了一部分就回调了，这就得在框架层面适配这个问题
    • tio-core内置UDP的支持
    • tio-websocket 发布到中心仓库，版本号：0.0.1-tio-websocket
    • tio-httpserver 发布到中心仓库，版本号：0.0.1-tio-http
